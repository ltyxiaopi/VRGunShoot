# Codex 交付记录

该目录用于存放 Codex 每次完成任务后的交付记录，供 Claude Code 审查和验收使用。

命名约定:
- `任务编号-任务名.md`
- 示例: `002-rifle-grab.md`
- 返工/追加记录可用后缀，如 `012-review-fixes.md`、`014-followup.md`

建议每份记录至少包含:
- **任务信息**：编号、标题、关联规格书路径
- **本次完成内容**：实现了哪些功能点
- **变更文件**：新增/修改的文件清单
- **验证结果**：在编辑器里跑了哪些自测、结果如何
- **Claude Code 重点检查项**：希望审查时特别关注的地方
- **已知说明**：占位、临时方案、遗留 TODO 等
