# VRGunShoot - 游戏设计文档

> 玩法方向由用户决定，Claude 可提建议和分析。本文档是需求源头，任务规格书从这里派生。

## 一句话定位
VR 步枪机械瞄具（觇孔 + 准星）射击打靶游戏，含训练打靶、城市攻防战、校枪三大模块。

## 瞄具术语（全项目通用，先统一）
- **觇孔**：后照门的圆孔（峰孔式），眼睛贴着看的那个。
- **准星 / 准信柱**：枪管前端的准星柱，柱顶用来压住靶心。
- **三点一线**：眼 → 觇孔 → 准星柱顶 → 靶心，对齐即十环。

## 已锁定的核心决策
1. **分析文本**：规则模板生成（离线、确定、可控），不联网调 AI。
2. **后座调整**：物理重新举枪瞄准 —— 每发后座让枪口上跳/偏移，玩家实际重新举稳、三点一线对回靶心。
3. **目标设备**：暂不定平台，先用 **XR Device Simulator** 在编辑器跑通玩法。
4. **射击方式**：训练 / 校枪 = 单发栓动（每发手动上膛）；攻防战 = 半自动。

## 模块一：训练场打靶
| 项 | 内容 |
|---|---|
| 瞄具 | 觇孔 + 准星，柱顶压靶心 = 十环 |
| 距离 | 100m / 200m 两种场景 |
| 目标 | 1.7m 高 标准胸环靶 |
| 弹数 | 5 发，单发栓动 |
| 后座 | 每打一发，枪偏一下（模拟后座）；玩家物理重新举枪、三点一线对回靶心 |
| 结算 | ① 评分：30–35 及格 / 36–44 良好 / 45+ 优秀；② 规则模板分析文本（如"弹着点偏右，应向左瞄准一点"）；③ 胸环靶图 + 你打的弹孔 |

## 模块二：城市攻防战
| 项 | 内容 |
|---|---|
| 形式 | 躲掩体后探身射击，半自动开火 |
| 目标 | **人（敌兵）**，随机出现：趴在掩体后 / 直线推进 / 斜向推进 |
| 瞄准 | 不需精瞄，直接打 |
| 合格 | 5 秒内全歼当波敌人 |

## 模块三：校枪（打 100m 靶、卧姿）
枪带一个**未知偏移**：准星柱顶压住靶心了，但弹着点不在十环（有固定偏差）。
流程：打 5 发 → 出胸环靶弹着图 → 填空算出修正量 → 应用修正 → 高亮调整后的瞄准点 → 显示距靶心距离 → 落在**半径 5cm 白圈**内 = 合格。

### 填空换算（准星管上下、觇孔管左右）
- **准星** ＿（顺 / 逆）时针旋转 ＿ **度**
  → 顺时针 360° = 弹着点**上移 24cm**；逆时针 360° = 下移 24cm（竖直方向）
- **觇孔** ＿（顺 / 逆）时针旋转 ＿ **格**
  → 顺时针 1 格 = 弹着点**左移 2cm**；逆时针 1 格 = 右移 2cm（水平方向）

> 校枪期间不叠加后座干扰（否则测不准固定偏移）。

## 核心系统（跨模块共用）
- **持枪与抓取**：双手持步枪（XRI Grab）。
- **机械瞄具**：觇孔 + 准星几何 + 三点一线判定。
- **弹道/命中**：射线开火 → 命中点 → 弹着点；命中计算预留可注入偏移 `zeroOffset`（校枪用）。
- **后座 + 栓动**：开火后枪偏移 + 单发上膛循环（攻防战为半自动）。
- **胸环靶 + 计分**：环区命中 → 环数；100m / 200m 配置。
- **结算 UI**：靶图 + 弹孔渲染 + 评分 + 分析文本。
- **模式管理**：主菜单选三模式入口。

## 玩家体验目标
真实的机械瞄具据枪体验 + 打靶成绩反馈 + 校枪的「调表对零」成就感。

## 美术风格
写实军事风（步枪、胸环靶、士兵、城市掩体）。素材以现成低面 3D 模型 + 贴图为主。

## 范围与优先级
- **MVP（先做）**：核心枪械系统（持枪/瞄具/弹道/后座栓动）→ 胸环靶计分 → 训练模式串联。
- **再做**：校枪模块。
- **后做**：城市攻防战（敌人 AI + 掩体场景）、主菜单。
